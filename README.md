## CONTEÚDOS ABORDADOS

### AWS CloudFormation
AWS CloudFormation é um serviço da AWS que permite criar, atualizar e excluir
recursos de infraestrutura de forma automatizada por meio de templates
declarativos escritos em **YAML** ou **JSON**. Esses templates descrevem os
recursos necessários e suas configurações, possibilitando a criação de ambientes
padronizados e reproduzíveis.

### Infraestrutura como Código (IaC)
Infraestrutura como Código é uma prática que permite definir a infraestrutura
por meio de arquivos de configuração, facilitando o versionamento, a automação
e o controle do ambiente ao longo do tempo.
